# receipt-digitization-data-entry
Converted physical retail receipt images into a structured Excel dataset, ensuring 100% accuracy in financial reconciliation and data organization.
## Project Overview
This project demonstrates precision and attention to detail in transforming unstructured physical data into a clean, searchable digital format. I processed multiple receipt images to create a centralized financial record.

## The Technical Process
Data Extraction: Manually transcribed line items, unit prices, and store metadata from various retailers like Walmart, Whole Foods, and Momi & Toy's.
Audit & Reconciliation: Verified that subtotals, tax calculations, and final totals in the spreadsheet matched the physical source images.
Format Standardization: Standardized shorthand item descriptions.
Data Privacy: Ensured sensitive payment details from the source images were redacted or excluded to maintain data security.

## Skills Demonstrated
Accuracy: High-precision data entry of alphanumeric SKU codes and Transaction IDs (TC#).
Information Organization: Created a logical schema including Date, Store Name, Item Detail, and Category.
Excel Management: Utilized spreadsheet tools to organize 30+ transactions across different global regions.
